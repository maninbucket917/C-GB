# C-GB

A basic Gameboy emulator project written in C.

## Compiling & Running

### Linux

Dependencies:
- make
- gcc
- SDL2 development libraries (libsdl2-dev)

To compile the source code, navigate to the root and run:  
```make```

To run C-GB with a specific ROM, run:  
```./bin/C-GB path/to/rom.gb```
